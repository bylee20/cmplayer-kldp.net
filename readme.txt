CMPlayer

CMPlayer는 통합/다중 자막을 지원하며, 여러가지 재생엔진을 백엔드로 이용하는 미디어 플레이어입니다.
현재 이용할 수 있는 재생엔진은 MPlayer와 xine입니다.

CMPlayer는 GNU GPL에 따라서 배포됩니다.

이용하시면서 불편한점이나 문제등이 있으신 경우 다음 중 한곳으로 연락 주시기 바랍니다.
